NVM을 이용한 Node.js 버전관리 (Windows용)

2019-01-17


NVM ( Node Version Manager ) - Windows용 공식페이지

NVM_Windows 다운로드


이글은 Windows 이용자를 위해 작성되었습니다.

어느세 Node.jS 버전이 10버전까지 올라왔습니다.

Mac OS에는 강력한 Node.js 버전관리 모듈인 N, NVM 이 있었지만

Window에서는 편리하게 사용할수있는 버전관리 모듈이 없었습니다 ㅠㅠ

하지만 nvm이 windows용이 나오면서 편하게 버전관리를 할수있게 되었습니다.

windows에서 이보다 편한 버전관리 모듈은 없다고 생각합니다.

이전 프로젝트는 Node는 v6이고 최근에는 v10으로 개발을 진행하게 되어 두가지 버전이상이 필요성을 느껴 글을 작성하게 되었습니다. 요즘에는 v6 지원하지 않는 모듈이 많습니다.

nvm 설치

설치전 제어판에 들어가서 Node.js 를 삭제시켜 주세요

Global로 설치한 모듈들도 같이삭제되니 모듈목록과, 버전을 따로 저장해두세요!!

버전관리를 해야되는데 Node.js 를 지우라니 이게무슨말이야?

의문이 드실껍니다. NVM 안에서 node를 재설치할 예정입니다. 가이드를 따라 진행해주세요

Window키 -> 제어판 -> 프로그램 및 기능 -> Node.js 오른쪽마우스 클릭후 제거해줍니다.

node.js 설치

이제 nvm을 통해 node.js를 설치해보겠습니다.

nvm에는 크게 3가지 기능이있습니다.

기능 명령어
node.js 특정버전 설치 nvm install v10.15.0
설치된 버전검색 nvm ls
node.js 사용할 버전선택 nvm use 10.15.0

그럼 이기능들을 가지고 node.js 설치와 교체를 진행해보겠습니다.

window키 + R 을 눌러 실행 창을 엽니다.

cmd 를 입력 후 확인을 눌러주세요.

우선 node.js 설치를 먼저 진행해보겠습니다.

화면에 nvm install v10.15.0 을입력해주세요 v 뒤에는 원하시는 버전을 입력하시면 됩니다.

이제 nvm ls 로 설치가 됐는지 확인을 해봅시다.

정상적으로 설치된것을 확인하실수 있습니다.

이제 사용할 노드버전을 적용해봅시다.

화면에 nvm use 10.15.0 입력해주세요

정상적으로 node가 설치되었습니다!!

node 버전이 나오지않을경우 터미널 재실행후 확인하세요

버전변경

위와 마찬가지로 nvm install 설치할버전 -> nvm ls -> nvm use 적용할 버전

이제 원하시는 node 버전을 선택해서 쓰시면됩니다!!

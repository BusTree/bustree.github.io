Node.js 소개 및 다운로드

Node.js 는 JavaScript 언어로 개발된 비동기이벤트처리 플랫폼입니다.

최근 웹서버 환경에서 자주 사용되는 강력한 언어입니다.


Node.js 다운로드 페이지

링크를 눌러 페이지 접속 후 LTS 버전을 다운받습니다.

설치확인

위에서 설치한 Node가 잘 설치되었나 확인해봐야겠죠?

window키 + R 을 눌러 실행 창을 엽니다.

cmd 를 입력 후 확인을 눌러주세요.

node -v 를 입력하여 node 버전을 확인합니다.

npm -v 입력하여 npm 버전을 확인합니다.
